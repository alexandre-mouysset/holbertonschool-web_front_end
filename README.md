# holbertonschool-web_front_end

## Setup rapide (nouveau PC)

### Prerequis
- Node.js (version LTS recommandee)
- npm

### Etapes
1. Cloner le repository

```bash
git clone <URL_DU_REPO>
cd holbertonschool-web_front_end
```

2. Installer les dependances

```bash
npm install
```

3. Lancer le serveur de developpement

```bash
npm run dev
```

4. Ouvrir la page dans le navigateur

```text
http://localhost:5500/0-index.html
```

## Notes
- Le dossier `node_modules` n'est pas versionne.
- Apres un `git clone`, il faut toujours lancer `npm install` avant `npm run dev`.
